# Rumus Cepat TPA

## Numerik

| No | Kategori | Rumus | Contoh |
|----|----------|-------|--------|
| 1 | Persen | x% dari y = (x × y)/100 | 20% dari 150 = 30 |
| 2 | Skala | JP = JS × Skala | 1:100.000, 2cm = 2km |
| 3 | Kecepatan | v = s/t | 100km/2jam = 50km/jam |
| 4 | Diskon | Akhir = Awal × (100% - diskon) | 100rb diskon 20% = 80rb |
| 5 | Persamaan | ax + b = c → x = (c-b)/a | 2x + 3 = 9 → x = 3 |
| 6 | Deret | Cari pola + - × ÷ pangkat | 2,4,6,8,... (+2) |
| 7 | Pecahan | a/b + c/d = (ad+bc)/bd | 1/2 + 1/3 = 5/6 |
| 8 | Rasio | a:b = c:d → a×d = b×c | 2:3 = 4:x → x=6 |
